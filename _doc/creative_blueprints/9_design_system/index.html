Design System

Modern Oriental Minimalism

Calm, crafted, premium, unhurried. One clear primary action per screen. Serif headers, soft shadows, restrained motion — never stark clinical white.

Color

Cream #FDFBF7
Brown #3E2723
Hong Tang Red #C62828
Warm grey #6F6258

Cream surface, brown content, red for the single most important action only. Tier metallics on tier surfaces.

Typography & Layout

Elegant serif display (~28–32pt) + clean sans body (~15–16pt). Tabular figures for crystals/prices. 8px grid, card containers, ~16pt padding. Sticky bottom CTAs above safe area. Soft corners (cards 16px, buttons 12px, pills full).

Core Components

Primary button Ghost Segmented Stepper − N + Chips Bottom sheet OTP boxes Status stepper

Calm eased motion (cross-fade tabs, spring sheets), micro-interactions (cart-badge bump, card tilt). Honors reduce-motion.

Accessibility & Localization

AA contrast, ≥44×44pt taps, dynamic type, screen-reader labels, no color-only signals (expiry = red + text). IDR "Rp 38.000", WIB/WITA/WIT, ~30% text-expansion headroom for future Bahasa Indonesia.
