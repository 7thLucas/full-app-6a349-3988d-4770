Design System

Modern Oriental Minimalism

Calm, crafted, premium, unhurried. One clear primary action per screen. Serif headers, soft shadows, restrained motion. Never stark, clinical white.

Refined serif display + clean sans body; tabular figures for crystals & prices. 8px grid, card containers, sticky bottom CTAs above the safe area. Soft corners (cards 16px, buttons 12px, pills full).

Primary button Ghost Segmented Stepper − N + Chips Bottom sheet OTP boxes Status stepper

Calm eased motion: cross-fade tabs, spring sheets, cart-badge bump, membership-card tilt. Honors reduce-motion.

Accessibility & Localization

AA contrast, ≥44×44pt taps, dynamic type, screen-reader labels, no color-only signals (expiry = red + text). IDR “Rp 38.000”, WIB/WITA/WIT, ~30% text-expansion headroom for future Bahasa Indonesia.
