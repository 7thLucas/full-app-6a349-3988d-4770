Scope, Quality & Roadmap

What ships, what waits, what must hold

Out of Scope (this version)

  • • Native in-app delivery — purchase/fulfillment stays on GoFood / GrabFood / ShopeeFood (those orders earn no crystals/Bowls). In-app "Delivery" = status display only.
  • • In-app social feed / community.
  • • User-generated reviews / ratings.
  • • Web ordering / desktop experience.

Roadmap Items

Native in-app delivery · Bahasa Indonesia localization (app is currently English).

Non-Functional Requirements

  • Performance: menu < 2s, 60fps on mid-tier, launch/Members'-Day traffic spikes.
  • Reliability: graceful network loss, cached menu, idempotent order placement.
  • Security: PCI DSS, TLS + at-rest encryption, OTP rate-limit/lockout, no on-device card data.
  • Platform: React Native / Flutter, iOS + Android, current + prior 2 OS versions.

Privacy & Compliance

Indonesia UU PDP (No. 27/2022): disclosed/consented data use, marketing-consent management, account/data deletion, data minimization. WCAG AA baselines. Consent-respecting analytics across the funnel.

Edge/empty/error states defined per the states contract for every data-driven screen — offline banners, OTP fallback, sold-out tags, voucher conflicts, payment retry (no double-charge), refund-pending notes.
