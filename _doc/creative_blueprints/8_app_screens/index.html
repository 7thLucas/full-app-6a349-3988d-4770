The App · Screen Map

Five tabs, one calm ordering loop

Persistent bottom nav: Home · Menu · Rewards · Cart · Me. Transactional screens open as bottom-up sheets; drill-downs push left-to-right.

① Home

Greeting + QR pass, hero banner carousel, Pickup/Delivery cards, loyalty snapshot, recent-orders strip.

② Menu

Store row + Pickup⇆Delivery toggle, scroll-spy category pills, cut-out item rows with Red "+".

③ Product Detail

Sheet: hero image, segmented Ice/Sugar, topping checklist with surcharges, live-price Add-to-Cart.

④ Cart & Checkout

Fulfillment + prep estimate, editable lines, voucher row, payment picker, price breakdown, Confirm Payment.

⑤ Order Status

Stepper Received → Preparing → Ready → Collected; pickup QR; live push + pull-to-refresh.

⑥ Rewards

Tilt-reflective membership card, Rewards Store grid + My Vouchers tabs, Sugar Crystals history.

⑦ Store Locator

Map with branded pins + draggable distance-sorted list, hours, directions, "Order from this store".

⑧ Me / Profile

Order history & reorder, personal info, addresses, payment methods, refer-a-friend, settings.

⑨ Onboarding / Auth

Country → phone → OTP (SMS/WhatsApp) → optional referral → permissions → minimal profile. Passwordless.

Every data-driven screen defines loading, populated, empty, and error states (skeletons first, never bare spinners).
